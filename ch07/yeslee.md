# 연산자(operator)

- 하나 이상의 표현식을 대상으로 산술, 비교, 논리, 타입, 지수 연산(operation)을 수행, 하나의 값을 만든다.
- 피연산자(operand): 연산의 대상으로 값으로 평가될 수 있는 표현식이다.

## 산술 연산자(arithmetic operator)

- 산술 연산이 불가능한 경우, NaN(Not-a-Number)를 반환한다.

### 이항(binary) 산술 연산자

- 피연산자 2개를 산술 연산하여 숫자 값을 만든다.

  ex) `+`, `-`, `*`, `/`, `%`

### 단항(unary) 산술 연산자

- 피연산자 1개를 산술 연산하여 숫자 값을 만든다.

  ex) `++`, `--`, `*`, `/`, `%`

- 증가/감소(`++`, `--`) 연산자는 피연산자의 값을 변경하는 부수 효과가 있다.

  - 전위 증가/감소 연산자: 피연산자 앞에 위치하며 피연산자의 값을 증가/감소시킨 후, 다른 연산을 수행한다.
  - 후위 증가/감소 연산자: 피연산자 뒤에 위치하며 다른 연산을 수행한 후, 피연산자의 값을 증가/감소시킨다.

- 숫자 타입이 아닌 피연산자에 `+` 단항 연산자를 사용하면 피연산자의 데이터 타입을 숫자타입으로 변환된다.

  ```javascript
  var x = "42";
  var y = true;
  var z = "JavaScript";

  console.log(+x); // 42
  console.log(+y); // 1
  console.log(+z); // Nan: 문자열은 숫자로 변환할 수 없다.
  ```

### 문자열 연결 연산자

- `+` 연산자는 피연산자 중 **하나 이상이 문자열인 경우 문자열 연결 연산자로 동작**한다.
- **암묵적 타입 변환(implicit coercion)**, **타입 강제 변환(type coercion)**: 개발자의 의도와 관련없이 자바스크립트 엔진에 의해 암묵적으로 타입이 자동 변환된다.

  ```javascript
  "4" + 2; // '42'

  1 + true; // 2: true는 1로 타입 변환된다.

  1 + null; // 1: null은 0으로 타입 변환된다.

  1 + undefined; // NaN: undefined는 숫자로 타입 변환이 되지 않는다.
  ```

  ```javascript
  var x = 1;
  ```

## 할당 연산자(assignment operator)

- 우항에 있는 피연산자의 평가 결과를 좌항에 있는 변수에 할당한다.
- 변수에 값을 할당하는 역할만 하지만 **할당된 값으로 평가**하므로 값으로 평가되는 표현식이다.

  ```javascript
  var x, y, z;

  // z - y - x 순으로 연쇄 할당이 진행되면서 각각 42로 평가된다.
  x = y = z = 42;

  console.log(x, y, z); // 42 42 42
  ```

## 비교 연산자(comparison operator)

### 동등(loose)/일치(strict) 비교 연산자

- 결과로 boolean 값을 반환한다.

| 연산자           | 예시      | 조건                          | 특징                                                                                                                |
| ---------------- | --------- | ----------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| 동등 비교 연산자 | x `==` y  | x와 y 값이 같은지?            | - 느슨한 비교<br>- **암묵적 타입 변환**을 통해 타입을 일치시킨 후 비교한다.                                         |
| 일치 비교 연산자 | x `===` y | x와 y 값과 **타입**이 같은지? | - 엄격한 비교<br>- `NaN`은 자신과 일치하지 않는 유일한 값이다. → `NaN` 여부는 `Number.isNaN`을 통해 확인할 수 있다. |

> 양의 0과 음의 0을 비교하면 true를 반환한다.
>
> ```javascript
> -0 == 0; // true
> -0 === 0; // true
> ```

### 대소 관계 비교 연산자

- 피연산자의 크기를 비교하여 boolean 값을 반환한다.

## 삼항 조건 연산자(ternary operator)

- 조건식의 평가 결과에 따라 반환할 값을 결정한다. → `조건식 ? true : false`

## 논리 연산자(logical operator)

- 논리합(||) 또는 논리곱(&&) 연산자 표현식의 평가 결과는 2개의 피연산자 중 어느 한쪽으로 표현된다.
- 논리 부정(!) 연산자는 boolean 값을 반환한다.

## 쉼표(,) 연산자

- 왼쪽 피연산자부터 차례로 피연산자를 평가하고 마지막 피연산자의 평가 결과를 반환한다.

## 그룹 연산자

- 소괄호 `()`로 피연산자를 감싸는 연산자로 연산자 우선순위가 가장 높다.

## typeof 연산자

- 피연산자의 데이터 타입을 문자열로 반환한다.

  ex) `string`, `number`, `boolean`, `undefined`, `symbol`, `object`, `function`

- **`null`은 `object`를 반환**하므로 null 타입 유무는 일치 연산자(`===`)를 사용한다.
- 선언하지 않은 식별자에 typeof 연산을 하면 `undefined`를 반환한다.

## 지수 연산자

- ES7에 도입되었으며 거듭 제곱을 수행한다.
  ```javascript
  4 ** 2; // 16
  ```
